# Marketing Positioning

## Core Line

```text
Generic agents make claims. FlowMemory agents can leave warranted receipts.
```

## Category Line

```text
Available is not enough. Warrantable is the standard.
```

## Simple Explanation

Most AI agents can act, call tools, and move through workflows.

FlowMemory asks the harder question:

```text
What happens when the agent makes a promise?
```

FlowMemory Warranted Agents turns that promise into:

- a PolicyCard;
- a bond-modeled warranty;
- required evidence envelopes;
- a deterministic runtime history;
- a FlowPulse-style memory artifact;
- a private scoped proof the user can carry.

## 60-Second Script

```text
AI agents are becoming workers.

They can research, code, transact, and act across the internet.

But most agents still ask users for trust.

FlowMemory changes the standard.

Generic agents make claims.
FlowMemory agents can leave warranted receipts.

The user defines the promise in a PolicyCard.
The agent quotes the work and puts a bond behind it.
The runtime records the machine history.
The evidence schema checks the receipt path.
FlowBond settles the outcome.
PulsePass gives the user scoped private proof.

If the agent closes the obligation, the warranty releases.
If payment happens without the required delivery path, the local model pays the user from the bond.

This is not just another agent log.
It is the start of warranted agent infrastructure.

Available is not enough.
Warrantable is the standard.
```

## Short Lines

- Agents with money on the line.
- Promises should have receipts.
- The user owns the rule. The agent bonds the promise.
- Agent work should be warrantable, not just available.
- FlowMemory turns agent promises into machine histories.
- A valid action surface is not always a valid machine history.

## Boundary

This language is for the local v0 architecture. It does not claim custody, wallet enforcement, production settlement, production adjudication, full privacy, zero-knowledge privacy, semantic truth, work-quality proof, or production verifier infrastructure.
